# Styld

Premium fashion platform for building and sharing outfits.

## Features

- **Upload clothing** from photos
- **Build outfits** using an intuitive carousel-based outfit builder
- **Browse catalog** of outfits from the community
- **Shop directly** from outfit details
- **Creator profiles** to explore individual styles

## Design

Dark minimalist aesthetic with Apple × fashion editorial influence. Calm, restrained, and confident design language.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- React Router
- Vite
- Modern CSS with CSS Variables
